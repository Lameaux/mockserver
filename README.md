**mox** is a mocking tool.

# About

It is written in Go with mappings defined in JSON.

# Installation

Make sure you have `GOPATH` set up correctly.

```shell
make install
```

# Usage

```shell
mox [flags]

--debug
--skipBanner
--port=8080
--adminPort=8081
--metricsPort=9090
```
